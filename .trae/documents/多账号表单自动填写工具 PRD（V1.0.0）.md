# 二、需求分析

## 2.1 业务背景

作为新媒体平台运营官，核心工作之一是协助博主账号对接品牌方合作，品牌方通过微信群发放各类表单（腾讯文档、石墨文档、问卷星、麦克CRM等）收集博主信息，运营官需将手中十几个博主账号的信息（粉丝量、阅读量、主页链接、合作报价等）逐一填写至对应表单中。当前纯手动填写模式，重复性工作极强，且多账号切换填写易出错、效率极低，亟需一款工具实现多账号信息的统一管理和表单自动填写，降低操作成本。

## 2.2 用户痛点

- 效率低下：需手动切换十几个博主账号，逐一填写至同一个/多个表单，重复性操作占用大量时间；

- 易出错：多账号信息（如报价、粉丝量）相近，手动填写易混淆、漏填、填错；

- 管理繁琐：十几个账号的信息分散存储（如表格、备忘录），修改、更新账号信息后，填写表单时需反复核对；

- 适配麻烦：品牌方使用的表单工具多样（腾讯文档、问卷星等），不同表单的填写格式、字段位置不同，需反复适应；

- 部署需求：优先需要本地部署，保障账号信息隐私安全，同时预留线上部署能力，便于后续灵活拓展。

## 2.3 需求核心

核心需求：实现多博主账号信息的统一管理，支持自动适配各类主流表单工具，一键完成多个账号信息的填写，减少重复性操作，提升效率；

次要需求：支持账号信息的增删改查、数据备份与恢复，记录填写日志，适配本地部署，预留线上部署接口；

潜在需求：支持表单字段的自定义匹配，批量导入导出账号信息，适配更多小众表单工具。

## 2.4 需求边界

### 2.4.1 纳入范围

- 账号管理：多博主账号信息（基础信息、运营数据、合作报价等）的增删改查、批量导入导出；

- 表单适配：适配腾讯文档、石墨文档、问卷星、麦克CRM、WPS表单、微信小程序报名工具等6类主流表单；

- 自动填写：支持选择单个/多个账号，一键自动填写至目标表单，支持表单字段与账号信息的自定义匹配；

- 数据管理：本地数据备份（手动备份、定时备份）、恢复，填写日志记录（填写时间、账号、表单、结果）；

- 部署支持：优先实现本地部署（Windows、Mac系统），完成线上部署相关开发（不进行实际部署，预留部署能力）；

- 基础设置：表单模板管理、账号分组管理、备份路径设置、定时备份时间设置。

- 账号安全校验：填表单过程中可能涉及微信账号的登录校验（需扫码登陆），扫码登陆成功一次之后，后续填写过程无需重复扫码登陆；

### 2.4.2 不纳入范围

- 表单创建功能：不支持创建新的表单，仅支持适配已有的品牌方表单；

- 品牌方对接：不涉及与品牌方的沟通、表单链接的收集，需用户手动输入表单链接；

- 线上部署落地：仅完成线上部署相关的开发工作（如接口适配、权限控制），不进行服务器部署、域名配置等实际操作；

- 小众表单适配：暂不支持除指定6类之外的小众表单工具，后续可作为迭代需求；

- 自动化抓取表单：不支持自动抓取微信群内的表单链接，需用户手动复制粘贴表单链接。

## 2.5 成功指标

- 效率指标：单个表单填写10个账号的时间，从手动1小时缩短至5分钟以内；

- 准确率指标：自动填写的信息准确率≥99%，减少手动填写错误；

- 适配指标：指定6类表单工具的适配成功率≥100%，填写过程无卡顿、无失败；

- 部署指标：本地部署成功率≥95%，Windows、Mac系统均可正常安装使用；

- 体验指标：操作流程简洁，账号管理、表单填写步骤不超过3步，无需复杂学习成本。

# 三、功能模块设计

整体功能分为6大核心模块，各模块独立运行且相互关联，优先实现核心模块，辅助模块同步推进，预留拓展接口。

## 3.1 账号管理模块（核心）

### 3.1.1 功能描述

实现多博主账号信息的统一管理，支持账号信息的全生命周期操作，便于用户快速维护、查询账号信息，为自动填写提供数据支撑。

### 3.1.2 核心功能点

- 账号新增：手动填写单个账号信息，字段包括（可自定义添加字段）：

    

    - 基础信息：博主姓名、账号昵称、账号类型（如抖音、小红书、视频号）、账号ID、主页链接；

    - 运营数据：粉丝量、平均阅读量/播放量、点赞量、评论量、粉丝画像（可选）；

    - 合作信息：合作报价（单条、套餐）、合作形式（图文、视频、直播）、是否接受置换、联系方式；

    - 备注信息：品牌方偏好、过往合作记录、特殊说明。

- 账号编辑/删除：支持修改已添加的账号信息，删除无效/过期账号，删除前需二次确认；

- 账号查询/筛选：支持按账号昵称、账号类型、粉丝量范围、报价范围筛选账号，支持关键词搜索；

- 账号分组：支持自定义分组（如按平台分组、按粉丝量分组），便于批量选择同类账号填写表单；

- 批量操作：

    

    - 批量导入：支持Excel/Csv格式导入账号信息，导入前需校验字段匹配度，提示错误数据；

    - 批量导出：支持将所有/筛选后的账号信息导出为Excel/Csv格式，便于备份、分享；

    - 批量修改：支持批量修改同一分组内账号的相同字段（如统一修改合作报价、联系方式）。

- 账号状态管理：支持设置账号状态（正常、暂停、过期），暂停状态的账号不可用于自动填写，便于管理无效账号。

## 3.2 表单适配与自动填写模块（核心）

### 3.2.1 功能描述

适配6类主流表单工具，实现表单字段与账号信息的自动匹配/自定义匹配，支持一键填写单个/多个账号信息至目标表单，核心解决重复填写痛点。

### 3.2.2 核心功能点

- 表单链接解析：用户手动输入表单链接（腾讯文档、石墨等6类），工具自动解析表单的字段名称、字段类型（输入框、下拉框、单选框等），显示解析结果；

- 字段匹配：

    

    - 自动匹配：工具根据表单字段名称（如“粉丝量”“账号主页”），自动匹配账号管理模块中的对应字段，无需手动操作；

    - 自定义匹配：若自动匹配失败（如表单字段名称自定义），支持用户手动选择表单字段对应的账号信息字段，保存匹配规则为模板，后续可复用；

    - 字段校验：解析表单字段时，校验字段是否为必填项，填写时提醒用户补充必填的账号信息。

- 自动填写：

    

    - 单个账号填写：选择单个账号，点击“一键填写”，工具自动将该账号信息填充至对应表单字段，填写完成后提示用户确认提交；

    - 多个账号填写：选择多个账号（或整个分组），点击“批量填写”，工具依次将每个账号信息填充至表单并提交（支持设置提交间隔，避免触发表单反爬）；

    - 填写异常处理：填写过程中若出现网络异常、表单链接失效、字段匹配错误等问题，立即停止填写，提示异常原因，并记录异常日志，支持重试；

    - 填写预览：填写前可预览账号信息与表单字段的匹配结果，确认无误后再执行填写操作，减少错误。

- 表单模板管理：将常用的表单链接、字段匹配规则保存为模板，后续无需重复解析、匹配，直接选择模板即可开始填写，支持模板的增删改查。

## 3.3 表单列表模块（核心）

### 3.3.1 功能描述

提供表单链接的分类管理能力，通过日期文件夹分组形式，方便用户回顾填写记录、快速定位目标表单，支持批量导入导出，提升表单管理效率。

### 3.3.2 核心功能点

- 日期文件夹管理：

    - 文件夹创建：默认按“YYYY-MM-DD”创建日期文件夹，支持自定义命名、重命名，文件夹图标以日期标识；

    - 文件夹操作：支持文件夹删除（删除前需二次确认，同步删除文件夹下所有表单链接）、拖拽排序；

    - 文件夹关联：表单链接创建/导入时需选择归属文件夹，支持批量移动表单链接至其他文件夹。

- 表单链接管理：

    - 单个添加：手动输入表单名称、表单链接，选择表单类型，关联至目标文件夹；

    - 批量导入：支持粘贴回车分隔的多个表单链接，自动校验链接有效性，批量添加至指定文件夹；

    - 批量导出：支持导出当前文件夹下所有表单链接（Excel/Csv格式，包含表单名称、链接、类型、添加时间）；

    - 链接操作：支持表单名称编辑、链接修改、单个/批量删除，显示表单添加时间、最后填写时间。

- 筛选与搜索：支持按表单类型、添加时间筛选，支持关键词搜索表单名称/链接。

## 3.4 数据管理模块（辅助）

### 3.4.1 功能描述

保障账号信息、填写日志、表单模板的数据安全，支持备份与恢复，便于用户追溯操作记录、找回丢失数据，适配本地部署的隐私需求。

### 3.4.2 核心功能点

- 数据备份：

    

    - 手动备份：用户可随时手动备份所有数据（账号信息、表单模板、填写日志、表单列表），选择本地备份路径（如电脑磁盘、U盘），备份格式为加密文件，保障隐私；

    - 定时备份：支持设置定时备份时间（如每天凌晨2点），自动备份数据至指定路径，可设置备份频率（每天、每周、每月），备份完成后提示用户。

- 数据恢复：支持选择本地备份文件，一键恢复数据（可选择恢复全部数据，或单独恢复账号信息、表单模板、表单列表），恢复前提示用户确认，避免覆盖现有数据；

- 填写日志：记录所有自动填写操作的日志，包括填写时间、表单链接、选择的账号、填写结果（成功/失败）、失败原因、提交次数，支持按时间、账号、表单筛选日志，日志可导出；

- 数据清理：支持清理过期的填写日志（如3个月前的日志），释放本地存储空间，清理前需二次确认，避免误删。

## 3.5 基础设置模块（辅助）

### 3.5.1 功能描述

提供工具的基础配置选项，适配用户的个性化使用习惯，保障工具稳定运行，为本地部署、后续线上部署预留配置接口。

### 3.5.2 核心功能点

- 通用设置：设置工具语言（中文）、主题（浅色/深色）、界面字体大小，支持重启后生效；

- 部署设置：

    

    - 本地部署设置：设置本地数据存储路径、备份路径，配置本地运行端口（默认端口可修改），查看本地部署状态；

    - 线上部署预留：预留线上部署配置接口（如服务器地址、端口、数据库账号密码），后续可直接配置开启线上部署模式。

- 填写设置：设置批量填写时的提交间隔（如1-5秒可调，避免触发表单反爬），设置必填字段校验提醒方式（弹窗/提示框）；

- 账号安全设置：设置工具登录密码（本地部署模式下，启动工具需输入密码，保障账号信息安全），支持修改密码、找回密码（通过预留邮箱）；

- 版本更新：支持查看工具当前版本，检测是否有更新（本地版本更新，无需联网更新核心功能），提示更新内容并引导更新。

## 3.6 帮助与反馈模块（辅助）

### 3.6.1 功能描述

为用户提供使用指导，解决使用过程中的疑问，收集用户反馈，便于后续迭代优化。

### 3.6.2 核心功能点

- 使用手册：内置详细的使用教程，包括账号管理、表单填写、表单列表管理、数据备份等操作的步骤说明，配有图文指引；

- 常见问题（FAQ）：整理使用过程中可能遇到的问题（如表单解析失败、自动填写失败、备份恢复失败、表单列表导入失败）及解决方案；

- 反馈提交：支持用户提交使用过程中的建议、bug反馈，填写反馈内容、联系方式，反馈记录可查看处理状态。

## 3.7 线上部署适配模块（预留）

### 3.7.1 功能描述

完成线上部署相关的开发工作，不进行实际部署，预留接口和配置项，便于后续需要线上部署时快速落地，保障线上数据安全与多端访问。

### 3.7.2 核心功能点

- 线上数据库适配：支持对接线上数据库（如MySQL），实现数据的线上存储与同步，与本地数据库兼容，可切换存储模式；

- 权限控制：预留线上用户权限管理接口，支持创建多用户（如管理员、普通用户），分配不同权限（管理员可管理所有数据，普通用户仅管理自己的账号信息）；

- 多端适配：预留Web端、移动端访问接口，线上部署后，支持多设备登录使用，数据实时同步；

- 线上安全配置：预留HTTPS加密、接口鉴权、数据加密存储等配置项，保障线上数据安全，防止信息泄露。

# 四、页面细节设计

页面设计遵循**简洁、易用、高效、可视化**的原则，贴合新媒体运营官的使用习惯，核心突出表单自动填写的核心流程，界面风格统一，支持浅色/深色主题切换，适配Windows、Mac不同屏幕尺寸。

## 4.1 整体页面布局

采用“侧边导航栏+顶部状态栏+主体内容区”的基础布局，主体内容区根据不同模块适配差异化的分割逻辑：

- 侧边导航栏：展示6大功能模块（账号管理、表单填写、表单列表、数据管理、基础设置、帮助与反馈），点击模块切换主体内容，当前选中模块高亮显示，支持折叠/展开（折叠后仅显示图标，展开显示图标+文字）；

- 顶部状态栏：显示工具名称、当前登录用户、当前时间、本地部署状态（如“本地部署已启动”），右侧提供主题切换、消息提醒、退出登录按钮；

- 主体内容区：不同模块适配差异化分割逻辑，核心操作按钮（如新增、填写、备份）固定在显眼位置，避免频繁查找，页面切换流畅且带平滑过渡动画。

## 4.2 核心页面细节

### 4.2.1 账号管理页面

- 页面顶部：搜索框（支持关键词搜索）、筛选栏（账号类型、粉丝量、状态）、批量操作按钮（导入、导出、批量修改、删除）、新增账号按钮；

- 页面主体：账号列表，展示账号昵称、账号类型、粉丝量、报价、状态、操作列（编辑、删除、查看详情），支持分页（每页10/20/50条可选），列表可排序（按粉丝量、添加时间）；

- 新增/编辑账号弹窗：字段分类展示（基础信息、运营数据等），必填字段标红，支持字段自定义添加，弹窗底部提供确认、取消按钮，提交前校验必填字段；

- 账号分组侧边栏：右侧可展开账号分组列表，点击分组筛选对应账号，支持新增、编辑、删除分组；

- 交互细节：账号列表hover时高亮，双击账号可快速进入编辑弹窗，批量操作后显示“操作成功，共处理X条账号”提示。

### 4.2.2 表单填写页面（核心页面）

核心为**左-中-右三列分割**布局，列宽支持手动拖拽调整（适配不同屏幕尺寸），整体占满主体内容区：

|---|---|---|---|

- 前端：使用npm进行依赖管理，package.json文件中明确依赖版本，锁定依赖版本（package-lock.json），避免版本冲突；
